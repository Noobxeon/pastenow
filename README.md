PasteNow — 轻量级 macOS 剪贴板管理器

PasteNow 是一个为 macOS 打造的 系统状态栏剪贴板管理器，支持历史记录、图片预览、固定、删除、全局快捷键、自动收回等功能。
界面简洁、无 Dock 图标、不打扰工作流。

适合希望提升效率的 macOS 用户。

⸻

✨ 功能特性
	•	✅ 状态栏常驻（无 Dock 图标，不占空间）
	•	✅ 历史剪切板记录
	•	✅ 支持文本 / 图片（图片显示大预览）
	•	✅ 右键固定 / 取消固定
	•	✅ 复制、删除一键操作
	•	✅ 支持搜索剪切板内容
	•	✅ 点击外部自动收回 Popover（控制中心风格）
	•	✅ 全局快捷键（⌘ + ⌥ + V）快速呼出
	•	✅ 支持深色模式 / 浅色模式
	•	✅ 完全本地，不上传任何数据


  功能   快捷键
  打开剪贴板窗口    ⌘ + ⌥ + V
  关闭剪贴板窗口   点击弹窗外区域
